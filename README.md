# Vending Machine

## How to start

#### 1. Open the terminal

#### 2. Clone repository

```http
  git clone git@github.com:joepotibutr/vending_machine.git
```

#### 3. Reach to the directory

```http
  cd vending_machine
```

#### 4. Install dependencies

```http
  npm install
```

#### 5. Start project

```http
  npm start
```
